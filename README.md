# SAB3 Bootstrap Project

This is a clean and responsive web layout created using **HTML**, **CSS**, and **Bootstrap 5**. The project showcases four astronomy-themed features with engaging visuals and smooth navigation.

##  Features

- Responsive design using Bootstrap grid system
- Hero section with a welcome message and "Get Started" button
- Four feature cards with images, titles, descriptions, and "Learn More" links
- Individual feature pages with custom backgrounds and welcome messages
- Smooth scroll from hero to features section

## Structure

- index.html – Main landing page
- feature1.html to feature4.html – Individual feature pages
- css/styles.css – Custom styling
- p1.jpg to p4.jpg – Feature images

## How to Run

1. Open index.html in your browser
2. Click “Get Started” to scroll to the Features section
3. Click “Learn More” on any card to view its dedicated page
